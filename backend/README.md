# Backend

### Структура проекта

```
mongo-init/
└──index.js — скрипт заполнения пустой бд
src/
├── __tests__/
│     └── server.test.js — запуск тестов
├── config/ — конфиг серверной части
├── database/ — подключение к базе данных
├── public/ — раздача админки
├── routes/ — роуты
├── middlewares/ — миддлевары
├── models/ — модели бекенда
└── controllers/ — контроллеры
```
